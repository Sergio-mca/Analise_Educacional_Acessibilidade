1. Introdução
 Este projeto tem como objetivo realizar uma análise exploratória sobre a infraestrutura das escolas
 brasileiras com base nos dados do Censo Escolar da Educação Básica 2024, fornecido pelo INEP. A
 análise busca identificar padrões, tendências e possíveis desigualdades regionais relacionadas ao acesso
 à tecnologia, conectividade e recursos de acessibilidade.
 2. Metodologia
 Para o desenvolvimento do projeto, utilizou-se a linguagem Python com a biblioteca PySpark, ideal para
 manipulação e análise de grandes volumes de dados. Os dados foram extraídos diretamente do arquivo
 oficial do Censo Escolar 2024, disponibilizado pelo Instituto Nacional de Estudos e Pesquisas Educacionais
 (INEP). As etapas incluíram: leitura e tratamento do dataset, criação de colunas derivadas, análises
 estatísticas e visuais, e exportação dos resultados para o Looker Studio.
 3. Etapas do Projeto
 • Seleção dos Dados: Utilização do microdados do Censo Escolar 2024 (INEP).
 • Tratamento e Limpeza: Remoção de inconsistências e padronização dos tipos de dados.
 • Criação de Colunas Derivadas: Indicadores de infraestrutura tecnológica e acessibilidade.
 • Análises Exploratórias: Cálculo de médias, correlações e detecção de outliers.
 • Visualizações: Gráficos de barras, linha, dispersão, boxplot e indicadores por estado.
 • Exportação: Geração de arquivo CSV tratado para integração com o Looker Studio.
 4. Resultados e Análises
 Os resultados indicaram variações significativas na disponibilidade de recursos tecnológicos e de
 acessibilidade entre os estados brasileiros. Observou-se que regiões mais desenvolvidas possuem maior
 percentual de escolas com acesso à internet e equipamentos digitais para alunos. Já estados com menor
 infraestrutura apresentaram índices reduzidos, indicando desigualdades educacionais que podem impactar
 a qualidade do ensino. A correlação entre recursos tecnológicos e de acessibilidade sugere que escolas
 mais equipadas tendem também a possuir melhores condições de inclusão.
 5. Conclusões
 A análise reforça a importância de políticas públicas voltadas à redução das desigualdades educacionais
 no Brasil. O uso de ferramentas como PySpark permite processar grandes volumes de dados e gerar
 insights relevantes para a gestão da educação. O trabalho demonstra, ainda, a capacidade de integrar
 ciência de dados e políticas educacionais para promover uma educação mais justa e acessível.
